## Installation

You may install `pawnote` from [NPM](https://npmjs.com/package/pawnote) using the package manager of your choice.

<Tabs>
  <TabItem label="pnpm">
    <Code code="pnpm add pawnote" lang="bash" />
  </TabItem>
  <TabItem label="yarn">
    <Code code="yarn add pawnote" lang="bash" />
  </TabItem>
  <TabItem label="npm">
    <Code code="npm install --save pawnote" lang="bash" />
  </TabItem>
</Tabs>
