## Installation

Vous pouvez installer `pawnote` depuis [NPM](https://npmjs.com/package/pawnote) en utilisant le gestionnaire de paquet de votre choix.

<Tabs>
  <TabItem label="pnpm">
    <Code code="pnpm add pawnote" lang="bash" />
  </TabItem>
  <TabItem label="yarn">
    <Code code="yarn add pawnote" lang="bash" />
  </TabItem>
  <TabItem label="npm">
    <Code code="npm install --save pawnote" lang="bash" />
  </TabItem>
</Tabs>
